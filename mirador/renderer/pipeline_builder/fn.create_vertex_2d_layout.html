pub fn create_vertex_2d_layout() -> VertexBufferLayout<'static>
Expand description

Create a vertex buffer layout for 2D positions.

This creates a vertex buffer layout for simple 2D vertex data consisting of [f32; 2] position arrays. The layout maps to shader location 0.

ยงMemory Layout

Each vertex consists of:

  • Position: vec2<f32> at shader location 0

ยงShader Compatibility

Use this layout with vertex shaders that have:

@vertex
fn vs_main(@location(0) position: vec2<f32>) -> ... {
    // vertex shader code
}
