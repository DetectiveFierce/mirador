Creates a primary button style with a mint green color scheme.

Primary buttons are used for the main action on a screen or form. They should be used sparingly - typically only one primary button per view.

ยงVisual Characteristics

  • Base Color: Dark mint green (rgb(30, 110, 30))
  • Hover State: Darker mint green for visual feedback
  • Pressed State: Darkest mint green to indicate active press
  • Disabled State: Muted light green to indicate unavailability
  • Typography: Medium weight, white text for high contrast

ยงDesign Rationale

The green color palette communicates positive action and success. The slightly desaturated tones provide a professional appearance while maintaining accessibility standards.
